## A&M Chat App 
A messaging program for sending text messages via Firebase
The program contains 7 activities, which are:
##### SignInActivity
##### SignUpActivity
##### MainActivity
##### InfoActivity
##### UsersActivity
##### ChatActivity
##### SearchActivity
